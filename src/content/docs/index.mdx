## Quick Start

```bash
# Create a new Zenthyr project
npx create-zenthyr-app my-app
cd my-app

# Start the development server
npm run dev
```

Features
Clojure Backend: Build robust server-side logic
React Frontend: Create modern user interfaces
Electron Integration: Package as native desktop apps
Hot Reloading: Seamless development experience
